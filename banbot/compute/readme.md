## 说明
此包下的所有指标是基于tick计算的，每个tick计算更新一步。并非向量化计算。
tick计算方式适用于高频交易，性能更好。(1s等)
向量化计算方式适合1m及以上维度

## 横盘震荡的评估
当前是否处于震荡行情，是基于时间周期和前面的行情幅度来衡量的。
1s的频繁趋势涨跌，在1分钟维度看可能就是震荡。
1s维度的频繁趋势涨跌，在最近120s内看是趋势规整的。但在500s内看就属于震荡行情。
故定义【震荡指数】如下：(观察20~30个即可) 
【震荡指数】   =  3窗口TrueRange  /  前500窗口的high-low
（这里不使用经典TureRange或ATR，原因是无法检测芝麻开花节节高行情的波动）

## 是否巨量的评估
NVol >= 15
NVol >= 10 and NVol >= roll(NVol, 5)
NVol >= 5 and NVol >= roll(NVol, 10)
